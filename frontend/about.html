About Our Hackathon Project

This project demonstrates a simulation of wind speed and thunderstorm prediction using machine learning models trained on historical weather data.

The backend is powered by Flask and continuously streams predictions from a validation dataset to mimic real-time weather forecasting.

On the frontend, we use Leaflet.js for map visualizations, where wind predictions and thunderstorm probabilities are displayed dynamically.

This hackathon solution can be extended to real-world scenarios by connecting to live weather APIs and integrating with alert systems.
